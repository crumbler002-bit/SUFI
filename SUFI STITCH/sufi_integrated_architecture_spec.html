# SUFI: The Integrated Intelligence Brain (Zustand Architecture) ## 🧠 1. Core State Definition (Zustand) The entire system now communicates through a single source of truth. ```typescript export const useSufiStore = create((set) => ({ intent: "", results: [], bestMatch: null, userVector: [], trends: [], mood: "neutral", loading: false, search: async (query) => { set({ loading: true, intent: query }); const res = await fetch("/api/search", { method: "POST", body: JSON.stringify({ query }) }); const data = await res.json(); set({ results: data.results, bestMatch: data.results[0], mood: data.mood, loading: false }); } })); ``` ## 🔌 2. Global Synchronization - **Hero Input:** Triggers the `search` action, updating the global intent and results. - **Discovery Grid:** Subscribes to `results` and re-ranks instantly with micro-FLIP animations. - **AI Concierge:** Subscribes to `intent` and `results` to provide RAG-enhanced, context-aware responses. - **Background (Three.js):** Subscribes to `mood` to shift colors (Romantic: Pink, Luxury: Violet, Fast: Green). - **Owner Dashboard:** Feeds from the same trend and booking signals as the user-facing app. ## 🔁 3. Continuous Feedback Loop 1. **User Input** -> 2. **Embedding & Ranking** -> 3. **State Update** -> 4. **UI Adaptation** -> 5. **User Learning**.

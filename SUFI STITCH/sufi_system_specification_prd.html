# SUFI: Smart Unified Food Intelligence - Project Specification ## 🎬 Global Vision SUFI is a motion-first intelligent system. It is defined by its responsiveness and predictive nature. Every interaction is a teaching moment for the platform. ## 🎨 Visual Identity & Theme - **Aesthetic:** Living, responsive, intelligent, predictive. - **Visual Cues:** Subtle pulses, glows, waves, glassmorphism, and fluid layout shifts. - **Tech Stack Focus:** GSAP (sequences), Framer Motion (layout), Lottie (feedback), Lenis (scroll). --- ## 🏗️ System Architecture ### 1. Landing System (The Story Engine) - **Hero Simulation:** - Input: "romantic dinner with wine" - Sequence: Focus expand -> AI Pulse -> Dynamic Result Fade-in -> Best Match Elevation. - **Discovery Demo:** Grid morphing and personalization highlights. - **AI Concierge Demo:** Word-by-word streaming, inline cards, and "Reserve" CTA triggers. ### 2. Discovery System (Intelligence Visualized) - **Behavior:** Input focus expansion, shimmer loading, and dynamic grid rearrangement. - **Logic:** "Rising" best matches and pulsing trending items. ### 3. Reservation System (High-Feedback Flow) - **Transition:** Card-to-page morph. - **Feedback:** Shimmer skeletons, glowing availability, and "locked-in" selection states. - **Success:** Lottie animation and sliding confirmation details. ### 4. Owner System (Real-Time Analytics) - **Live Updates:** Booking pings and calendar refreshes. - **Visuals:** Animating SVG/Canvas growth lines and instant list updates. --- ## 🎬 Motion Requirements - **Transitions:** Layout-neutral morphs (Framer Motion `layoutId`). - **Feedback:** Physics-based springs for "premium" feel. - **Atmosphere:** Three.js background with subtle "intelligence" particles.
